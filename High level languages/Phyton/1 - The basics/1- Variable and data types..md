# Data types :
## Numeric types :
- int() : for whole numbers
- float() : for decimal numbers
- complex() : for complex numbers
## Text type :
- str() : (string) for a sequence of character
## boolean type :
- bool() : for things either true or false

# Type casting :
used to transform a datatype of a variable to another one.
example : 1 = 2(3)
     1 is the new variable.
     2 is the data type we wanna transform the old variable to.
     3 is the old variable
